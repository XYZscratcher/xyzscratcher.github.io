简介

jian.css 是一个超轻的 CSS 框架，拥有以下特点：

  • 🎈️超级轻量，比 water.css、simple.css 等 CSS 框架小得多。
  • 📊️高兼容性，支持大多数主流浏览器，包括 IE9（使用兼容版本）。
  • ✨️效果出众，丝毫不亚于 Bootstrap 等“重量级”框架（个人看法）。
  • 🎨️高自定义性，只用一行 JavaScript 代码，将主题颜色自由切换（使用普通版本）。
  • 📏️使用黄金比例，让您的网站更易阅读。
